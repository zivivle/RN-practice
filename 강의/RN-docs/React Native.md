### React Native 장점
- 하나의 코드로 ios, 안드로이드, 웹까지 대응이 가능하다
- 러닝커브가 높은 리액트 사용이 가능하다
- 코드 푸시로 빠른 업데이트가 가능하다 (가장 큰 장점)
- Fast Refresh -> 불필요한 새로고침이 필요없음
- 오픈소스 플랫폼 -> 큰 커뮤니티 존재

### React Native 단점
- 일부 기능 Native에 대한 지식이 필요하다
- 오픈소스, 라이브러리 의존도가 생길 수 있다.
- 잦은 업데이트!
