## 1. ?? 연산자 사용
![alt text](image-4.png)
: 앞에 값이 존재하지 않을 경우 ?? 연산자를 통해 뒤에 올 값으로 대체되도록 한다
```js
undefined ?? null // -> null
{ ... } ?? null // -> { ... }
```
